# Contributing to print-svc

## Prerequisites
.NET 8 SDK / Windows 10/11 / Docker Desktop / a printer

## Getting started
git clone -> cp appsettings.example.json -> dotnet run

## Branch naming
feat/... fix/... refactor/... chore/... docs/...

## Commit messages
Conventional Commits : imperative, lowercase, under 72 characters

## Workflow
branch -> code -> dotnet test -> PR -> review -> merge

## Running tests
dotnet test

## Code style
.editorconfig - 4 spaces, braces new line, explicit types
